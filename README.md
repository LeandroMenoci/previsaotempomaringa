# Previsão do Tempo Android

Aplicativo Android desenvolvido em Java para exibir previsão do tempo, mapa com marcador da cidade e troca de cidade via QR Code.

## Funcionalidades
- SplashScreen com logo
- TabBar com duas abas: Previsão e Mapa
- Lista de previsão do tempo via API HG Brasil
- Mapa com marcador fixo
- Menu na AppBar com tela "Sobre"
- Troca de cidade via QR Code (ZXing)

## Tecnologias
- Java
- Android Studio
- Retrofit
- Google Maps API
- ZXing QR Code Scanner
